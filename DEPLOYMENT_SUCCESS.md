### 🔧 技术架构

#### 前端服务

- **框架**: React 18 + TypeScript + Vite
- **样式**: Tailwind CSS + 自定义动画系统
- **路由**: 自定义 RouterContext (已修复导航问题)
- **状态管理**: Zustand
- **Web3 集成**: Ethers.js + 已部署的 Sepolia 合约

#### 后端服务

- **AI 服务**: Python FastAPI (端口 8001)
- **代理服务器**: Node.js Express (端口 3000)
- **区块链**: Sepolia 测试网络
- **合约地址**: `0x6af445EA589D8f550a3D1dacf34745071a4D5b4F`
