### 2. 智能钱包连接逻辑 🔗

#### 优化前问题

- 每次点击按钮都会返回登录界面
- 用户体验不连贯

#### 优化后体验

- ✅ **首次访问**: 显示精美的数字星图登录封面
- ✅ **后续访问**: 直接进入应用，无需重复连接
- ✅ **状态持久化**: 使用 localStorage 记录连接历史
- ✅ **开发模式**: 提供重置连接状态的调试按钮

#### 技术实现

```typescript
// 检查是否曾经连接过钱包
const [hasEverConnected, setHasEverConnected] = useState(false);

useEffect(() => {
  const everConnected =
    localStorage.getItem("sheGuardian_hasConnected") === "true";
  setHasEverConnected(everConnected);
}, []);

// 只在从未连接过钱包时显示登录封面
if (!hasEverConnected) {
  return <WalletConnectCover />;
}
```
