# 🎯 Emergency Guardian Demo 展示使用指南

## 📋 准备工作

### 1. 启动开发服务器

```bash
cd frontend
npm run dev
```

确保服务器运行在 `http://localhost:5173/`

### 2. 清除旧数据（如果遇到错误）

- 打开 `frontend/clear-storage.html`
- 点击"清除本地存储"按钮

## 🚀 Demo 展示方式

### 方式一：使用演示展示页面（推荐）

1. **打开演示页面**

   ```
   打开文件：frontend/demo-showcase.html
   ```

2. **选择演示模式**
   - 🏠 **用户仪表板** - 体验受保护用户视角
   - 👨‍⚕️ **监护人控制台** - 体验监护人审批视角
   - 🚨 **紧急请求流程** - 完整端到端流程演示

### 方式二：直接访问主应用

1. **访问主应用**

   ```
   http://localhost:5173/
   ```

2. **使用角色切换**
   - 右上角有"切换角色"按钮
   - 可以在用户和监护人角色间切换

## 🎬 完整演示流程

### 第一步：展示系统概览

1. **打开用户仪表板**

   - 展示系统统计数据
   - 查看现有的紧急请求
   - 介绍界面布局

2. **关键展示点**：
   - 📊 统计卡片：总紧急情况、活跃监护人、保护金额
   - 📝 最近活动：系统活动日志
   - 👥 监护人状态：当前监护人信息

### 第二步：创建紧急请求

1. **点击"紧急求助"按钮**

   - 位置：右上角红色按钮 或 仪表板中的大按钮

2. **填写紧急请求表单**

   ```
   紧急类型：选择"医疗紧急"
   紧急级别：选择"高"
   标题：输入"紧急医疗费用"
   描述：输入详细描述
   请求金额：输入"3.5"
   接收地址：可以使用默认地址
   ```

3. **提交请求**

   - 点击"下一步"完成表单
   - 最后点击"提交紧急请求"
   - ✅ **现在可以成功创建请求了！**

4. **观察结果**
   - 查看成功通知
   - 注意 URL 变化和页面更新

### 第三步：监护人审批流程

1. **切换到监护人角色**

   - 点击右上角的"用户"按钮
   - 切换为"监护人"角色

2. **查看监护人控制台**

   - 观察界面变化
   - 查看"待审批请求"部分
   - 展示刚创建的紧急请求

3. **选择监护人身份**

   - ✅ **新功能：在"待审批请求"卡片中可以选择当前监护人**
   - 下拉菜单选择：张医生、李律师、或王财务
   - 每个监护人都可以独立进行审批操作

4. **执行审批操作**

   - 找到刚创建的紧急请求
   - 选择第一个监护人（如：张医生）
   - 点击"批准"按钮
   - 观察状态变化

5. **切换监护人继续审批**

   - 在下拉菜单中选择第二个监护人（如：李律师）
   - 再次点击"批准"按钮
   - ✅ **现在需要多个监护人批准才能执行！**

6. **查看审批结果**
   - 查看通知更新
   - 观察活动日志变化
   - 展示多重签名进度
   - 当所有监护人都批准后，状态变为"已执行"

### 第四步：展示系统特性

1. **数据持久化测试**

   - 刷新页面
   - 验证所有数据都保持不变
   - 展示本地存储功能

2. **实时状态更新**

   - 展示审批后的状态变化
   - 查看通知系统
   - 观察活动日志更新

3. **多角色体验**

   - 在用户和监护人角色间切换
   - 展示不同视角的界面差异
   - 演示权限控制

4. **多监护人审批**
   - ✅ **演示多重签名机制**
   - 展示不同监护人的审批状态
   - 只有获得足够监护人批准才能执行

## 🔧 常见问题处理

### 如果页面空白

1. 打开 `frontend/clear-storage.html`
2. 清除本地存储
3. 重新打开主应用

### 如果没有数据

- 系统会自动加载演示数据
- 如果没有，刷新页面即可

### 如果导航不工作

- 确保使用最新的代码
- 检查浏览器控制台是否有错误

### 如果创建请求失败

- ✅ **已修复：现在不需要连接钱包即可创建演示请求**
- 系统会自动创建模拟的区块链交易

### 如果只能看到一个监护人

- ✅ **已修复：现在可以在监护人控制台中切换不同的监护人身份**
- 使用"当前监护人"下拉菜单选择不同监护人
